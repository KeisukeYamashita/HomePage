## 開発用サイトについて

[本サイト](https://takajodormitory.github.io/HomePage/index.html)は、[高城寮のホームページ](https://www.tokuyama.ac.jp/dormitory/)の開発用のサイトです。  
このリポジトリでサイトを開発し、GitHub Pages を使って開発中のサイトを公開します。

## ディレクトリ構成

```
HomePage
├───assets
│   ├───images
│   │   └───Facility
│   ├───json
│   ├───movies
│   ├───scripts
│   └───styles
└───pages
    ├───english
    ├───endeavors
    └───notice

```
